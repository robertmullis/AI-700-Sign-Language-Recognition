# AI-700-Sign-Language-Recognition

## Description

This project aims to construct a CNN model capable of high-accuracy static American Sign Language character recognition. This is a term project for AI-700: Deep Learning.

## Data

This project uses the [Sign Language MNIST](https://www.kaggle.com/datasets/datamunge/sign-language-mnist) dataset from Kaggle. 

To run this project:
1. Download the dataset from Kaggle.
2. Unzip it and place the `sign_mnist_train.csv` and `sign_mnist_test.csv` files into a folder named `data/` in this directory.
